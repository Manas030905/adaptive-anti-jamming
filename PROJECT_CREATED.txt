## 🎯 CORE FEATURES IMPLEMENTED

### ✅ Signal Processing Module
- Load WAV files (auto converts stereo → mono)
- Generate test signals (multi-frequency)

- Sample rate handling

### ✅ Jammer Generation
- **Tone Jammer:** Single frequency (180 Hz)
- **White Noise:** Broadband random noise
- **Sweep Jammer:** Linear frequency sweep (50-500 Hz)
- **Burst Jammer:** High-energy intermittent pulses
- **Combined Jammer:** Realistic mix of all types

### ✅ Adaptive Filtering
- **LMS (Least Mean Squares)**
  - Gradient-based learning
  - Parameter: μ (step-size)
  - Complexity: O(N) per sample
  
- **NLMS (Normalized LMS)**
  - Adaptive normalization by input power
  - Parameter: μ (normalized, larger values allowed)
  - Better convergence than LMS
  
- **RLS (Recursive Least Squares)**
  - Optimal in least-squares sense
  - Fast exponential convergence
  - Parameters: λ (forgetting factor)
