## 🎯 JAMMER TYPES

1. **Tone Jammer** - Single frequency (180 Hz narrowband)
2. **White Noise** - Broadband random noise
3. **Sweep** - Linear frequency sweep (50-500 Hz chirp)
4. **Burst** - High-energy intermittent pulses
5. **Combined** - Realistic mix of all types

## 🔧 CUSTOMIZATION

No programming needed! Just edit `config.py`:

```python
# Adjust filter parameters
FILTER_CONFIG['filter_order'] = 128      # Default: 64
FILTER_CONFIG['lms']['mu'] = 0.02         # Default: 0.01
FILTER_CONFIG['nlms']['mu'] = 0.7         # Default: 0.5
FILTER_CONFIG['rls']['lambda_param'] = 0.995  # Default: 0.99

# Change jammer type
JAMMER_CONFIG['primary_type'] = 'tone'    # Default: 'combined'

# Adjust signal properties
AUDIO_CONFIG['duration'] = 5.0             # Default: 3.0
CHANNEL_CONFIG['input_snr_db'] = 20.0     # Default: 15.0
```

## ⚡ EXECUTION PIPELINE

```

    ↓

    ↓
[8] Generate Visualizations
    ↓

    ↓
```
